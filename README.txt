  Branded Drupal Adminitration Theme
======================================

Branded is a very clean and modern Drupal 7 administration theme that focuses on
keeping the interface simple and friendly as well as offering options for branding.

Users can brand the theme in the following ways:

1. A logo can be added to the right of the header.
2. A logo can be added to the left of header.
3. A footer text can be configured
